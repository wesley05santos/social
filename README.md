# README
```
Projeto para estudos em Ruby on Rails
Utilizando tecnologia Turbo,
dentro das variáveis, Turbo_Frame e
Turbo_Stream.
Além disso iniciando estudos também
com JavaScript para criar algumas estilizações
dinâmicas dentro do projeto.

O projeto em si consiste em uma simulação
de rede social, onde usuários fazem login e
criam artigos. Podem dar Follow e Unfollow em outros
usuários, localizando estes usuários através de busca.
Além disto pode também abrir um chat com usuários que segue
e é seguido por ele.

A tecnologia do chat, funciona também com websocket
atualizando em tempo real a página de conversação.
```
